# Listee Web

This web client for listee currently runs on <https://pizza.bartolomey.me/>.
It's a vuejs app currently using the v1 API of Listee <https://github.com/aix-pizza-time/listee-api>

## Running

```
yarn install

# For development
yarn run serve 
yarn run lint
# For production
yarn run build
docker build -t ... ./
```
